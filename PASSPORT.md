# Паспорт репозитория: pro2

## Что это

Нейросемантическая система с Q6-пространством — 6-мерным бинарным пространством смыслов, отображённым на 64 гексаграммы И-Цзин. Реализует замкнутый цикл обучения: граф знаний → порядок обучения → генерация → адаптация → новые пробелы.

## Какую грань реальности исследует

**Семантический угол**: как смыслы располагаются в пространстве, как связаны между собой, как модель обучается и заполняет пробелы знаний.

## Нативный формат

- **Единица хранения**: Концепт с Q6-координатой `[b0, b1, b2, b3, b4, b5]`
- **Пространство**: 6D бинарное → 64 состояния → 64 гексаграммы
- **Объём**: ~40 концептов, 4 домена (физика, химия, биология, математика)
- **Файлы данных**: `bidir_train_v2_log.json`, `checkpoint_bidir_v2.pt`

## Чем полезен другим репозиториям

**Даёт:**
- Семантические эмбеддинги концептов (cosine similarity)
- Q6-координаты для любого текста
- Граф знаний с весами связей
- Метрику "закрытия пробела" между концептами
- Корпус размеченных текстов по доменам

**Принимает:**
- Размеченные документы с доменом (из info1)
- CA-паттерны для символьной верификации (из meta)
- Теорию трансформации знаний K₀→K₁→K₂ (из data7)
- Любой текст для классификации по Q6

## Мосты к другим репо

| К репо | Тип моста | Механизм |
|--------|-----------|----------|
| **info1** | Методологический | α-уровни ↔ глубина Q6-пространства |
| **meta** | Символьный | Q6-координата ↔ номер гексаграммы ↔ CA-правило |
| **data7** | Теоретический | knowledge_transformation_theory ↔ bidir_train.py (реализует НЕДОСТАЮЩУЮ ПЕТЛЮ) |
| **nautilus** | Портальный | `Pro2Adapter` + `Data7Adapter` в `portal.py` |

### Конкретные соответствия:

```
info1 α=+4 (Онтология)    ↔  pro2 Q6 dim=5 (самый абстрактный)
info1 α=0  (Концепция)    ↔  pro2 Q6 dim=2-3 (средний уровень)
info1 α=-4 (Код)          ↔  pro2 Q6 dim=0 (самый конкретный)

meta CA-правило N         ↔  pro2 Q6[bin(N % 64)]
meta гексаграмма [50] 鼎  ↔  pro2 концепт "fire/transform"

data7 Диссертации         ↔  pro2 специализированный корпус (глубокий домен)
data7 Энциклопедии        ↔  pro2 обобщённые концепты (PageRank-центры)
data7 K₀→K₁→K₂           ↔  pro2 bidir_train.py замкнутый цикл
data7 MISSING LOOP        ↔  pro2 GPT→QFilter→AdaptiveLearning→gaps→hypotheses
```

## Ключевые алгоритмы

- **Bidirectional Training** (`bidir_train.py`): замкнутый цикл ⇑⇓ (реализует НЕДОСТАЮЩУЮ ПЕТЛЮ из data7)
- **KnowledgeGraph** (`bidir_train.py`): граф с PageRank-центральностью и TSP-порядком обхода
- **AdaptiveLearning**: динамический выбор следующего концепта по пробелам (↔ domain_triplet_loss)
- **QFilter**: фильтрация качества текстов 0.0..1.0
- **HypothesisGenerator**: генерация гипотез из пробелов знаний (↔ self_dialog stage 3)
- **Self-Improvement**: итеративные циклы с метриками CD/VT/CR/DB

## Как подключиться к Nautilus Portal

**Уровень 0** — уже подключён (`Pro2Adapter` в `portal.py`)

**Уровень 3** — полный адаптер:
```python
from portal import Pro2Adapter
from nautilus.core import NautilusPortal

portal = NautilusPortal()
portal.register("pro2", Pro2Adapter())
result = portal.query("crystal")
```

**Экспорт корпуса для других репо:**
```bash
python portal.py --json --query "all" > pro2_corpus.json
```
