# Паспорт репозитория: meta

## Что это

Система символьной верификации на базе клеточных автоматов (CA). 256 правил Вольфрама образуют пространство паттернов — от простейшего порядка (правило 0) до хаоса (правило 30) и сложности (правило 110). Отображение: 256 CA-правил → 64 гексаграммы → Q6-координаты. **Символьный детектор паттернов.**

## Какую грань реальности исследует

**Символьный угол**: как структурные паттерны проявляются на разных масштабах — в клеточных автоматах, в гексаграммах И-Цзин, в семантических пространствах. Правило порождает форму; форма несёт смысл.

## Нативный формат

- **Единица хранения**: CA-правило (целое число 0..255)
- **Пространство**: 256 правил × {детерминированный, случайный, сложный} класс
- **Символьный слой**: 64 гексаграммы как сжатое представление (rule % 64 + 1)
- **Q6-слой**: 6 бит = прямая Q6-координата

### Схема пространства CA:

```
Класс I   — правила с простым порядком    (правила 0, 8, 32...)
Класс II  — периодические паттерны        (правила 4, 12...)
Класс III — хаотичные                     (правило 30, 45, 73...)
Класс IV  — сложное поведение / граница   (правило 110, 124... ← вычислимость)
```

## Чем полезен другим репозиториям

**Даёт:**
- Символьную верификацию: концепт из pro2/info1 → CA-класс → интерпретация
- 256-мерное пространство паттернов как "словарь форм"
- Гексаграммы как медиаторы между CA и Q6
- Детектор "хаос / порядок / граница" для любого паттерна

**Принимает:**
- Q6-координаты из pro2 (для интерпретации через гексаграммы)
- α-уровни из info1 (для определения CA-класса уровня абстракции)

## Мосты к другим репо

| К репо | Тип моста | Механизм |
|--------|-----------|----------|
| **pro2** | Структурный | CA-правило N → Q6-биты `bin(N % 64, 6)` |
| **info1** | Аналогический | CA-класс ↔ α-уровень (класс IV ↔ α=0, граница) |
| **nautilus** | Портальный | `MetaAdapter` в `portal.py` |

### Соответствие CA ↔ Гексаграмма ↔ Q6:

```
CA-правило 49  →  гексаграмма [50] 鼎 Котёл  →  Q6[1,1,0,0,0,1]
CA-правило 30  →  гексаграмма [31] 咸 Взаимодействие → Q6[0,1,1,1,1,0]
CA-правило 110 →  гексаграмма [47] 困 Истощение → Q6[1,0,1,1,1,0]
CA-правило 0   →  гексаграмма [1]  乾 Творчество → Q6[0,0,0,0,0,0]
CA-правило 255 →  гексаграмма [64] 未濟 До завершения → Q6[1,1,1,1,1,1]
```
